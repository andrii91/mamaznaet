Тренинг для родителей детей от 0 до 18 лет

Система Осознанного Воспитания

Освойте комплексный подход к развитию характера ребенка, чтобы воспитать самостоятельную и уверенную личность, избежав детских истерик и стресса

2000+ счастливых мам прошли программу

Принять участие

Это легендарный тренинг, который прошло более 2000 семей. В основе тренинга лежат методики и инструменты, которые помогают сформировать характер ребенка.

Характер – это главное качество успешного, счастливого человека, который может выбраться из любой ситуации, обучиться любому навыку и построить счастливую жизнь

Программа тренинга состоит из 4-х ступеней

Ступень 1. Формируем свою систему воспитания. Закладываем основу для здоровых привычек в поведении.

Основные уроки ступени:
  1. Эмоциональное развитие ребенка.
  2. Нормы здорового сна и питания для детей.
  3. Внедрение распорядка дня.
  4. Самостоятельное засыпание для детей.
  5. Снятие негативных привычек в поведении.
Дополнительные уроки:
  1. Свои точки тревожности, снятие напряжения и стресса.
  2. Экспресс-методы успокоения себя в любой ситуации.

Ступень 2. Учим правильному общению, развиваем навыки.

Основные уроки ступени:
  1. Бытовая самостоятельность: еда, гигиена.
  2. Формулировки просьб, требований и указаний.
  3. Замечания, критика и обратная связь.
  4. Контроль эмоций: нервные всплески, плохое настроение.
  5. Снятие агрессии, грубости и хамства, переход в уважительную форму общения.
  6. Самостоятельное общение с окружающим миром.
Дополнительные уроки:
  1. Диалог для развития эмпатии у детей.
  2. Снятие страхов, мешающих сну и отдыху.
  3. Детский обман: как реагировать и справляться.
  4. Детские войны: снимаем противостояния между братьями и сестрами.
  5. Беседа примирения. Алгоритм примирения с ребенком.
  6. Техники извинения и прощения.

Ступень 3. Учим правильному общению, развиваем навыки.

Основные уроки ступени:
  1. Определение правил семьи.
  2. Внедрение и закрепление правил, корректировка и привыкание
  3. Контроль выполнения правил для всей семьи.
  4. Алгоритм построения договоренностей.
  5. Правила выполнения уроков
  6. Мышление достижения и результата.
Дополнительные уроки:
  1. Формирование стремлений и желаний.
  2. Учебная самостоятельность: выполнение уроков, контроль занятий.
  3. Персональные домашние обязанности.
  4. Гаджеты, игры, развлечения.

Ступень 4. Готовим к взрослой жизни, развиваем ответственность и уверенность.

Основные уроки ступени:
  1. Постановка цели для ребенка.
  2. Самостоятельные решения и способность делать выбор.
  3. Самооценка ребенка, его личное восприятие.
  4. Развитие принципы ребенка, развитие самоуважения.
  5. Сексуальное воспитание подростка.
  6. Финансовая грамотность для ребенка.
Дополнительные уроки:
  1. Внешние преобразования.
  2. Персональные достижения, личная шкала успеха.
  3. Нематериальная мотивация.
  4. Финансовая мотивация.
  5. Противостояние буллингу, травле, токсичным отношениям.
